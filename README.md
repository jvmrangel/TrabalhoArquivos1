# TrabalhoArquivos1
Busca Binária de CEP - Professor: Renato Mauro

1. Baixar projeto do GIT
2. Adicionar o arquivo "cep_ordenado.dat" à pasta "TrabalhoArquivos1"
3. Executar o programa utilizando o terminal com o seguinte comando:
	> ./BuscaCEP [CEP]
	* Onde [CEP] é o cep que deseja consultar.
